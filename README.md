# GifTastic
*User will have 6 ready buttons with content of different animated TV shows.* 
Each button will generate **10 gifs** from **GIFY API.**
User have an option to add the button of his choice. 

![](assets/img.png)


Built with:
1. HTML
1. CSS
1. JavaSript
1. Jquery
1. AJAX
1. CSS Animate
1. Font Awesome
